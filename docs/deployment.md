# Deployment notes

This project is designed as a **Next.js** example.

## Local
```bash
npm install
npm run dev
```

## Production
Recommended targets:
- Vercel
- self-hosted Node runtime for Next.js

## Vercel
1. Push repo to GitHub
2. Import into Vercel
3. Use default Next.js settings
4. Deploy

## Notes
- The project uses the App Router.
- The dev command uses Turbopack.
- No Babel-specific configuration is required.
